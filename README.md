# Statistical Models - University Project

## 📌 Overview
This repository contains a university project developed for the **Statistical Models** exam. 

## 📊 Dataset
The analysis utilizes a publicly available dataset (Rotterdam Breast Cancer cohort).

## 📂 Contents
* `rizzo_031064.ipynb`: The complete Jupyter Notebook with the source code and analysis.
* `Predictive Modeling and Causal Inference for Prognostic Stratification in the Rotterdam Breast Cancer Cohort.pdf`: The comprehensive project abstract detailing the background, methodology, and results.
